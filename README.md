# Augusta-Richmond-County-Urbanization
Here's a quick little ArcGIS analysis of land usage change in Augusta-Richmond County, Georgia, from 1992 to 2020. 	Using the ESA Climate Change Initiative's Global Land Cover 1992-2020 dataset, this report displays which zip codes contained the greatest proportion of land converted to urban usage from 1992 to 2020. 

These results indicate that the 30909 zip code in northwest Augusta has experienced the greatest proportion of urbanization, while southwest and southeast Augusta remain fairly untouched.
